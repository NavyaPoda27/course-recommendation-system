# Intelligent Course Recommendation System
## Overview
This project recommends online courses based on a user's:
- Skills
- Interests
- Career Goals
The recommendation engine uses:
- TF-IDF Vectorization
- Cosine Similarity
- Rating-based Ranking
Datasets used:
- Coursera
- Udemy
- edX
## Technologies Used
- Python
- Pandas
- Scikit-Learn
- NLP
- TF-IDF
## Sample Input
Skills:Python, SQL
Interest: Data Science
Goal: Data Analyst
## Sample Output
1. IBM Data Analyst
2. Google Data Analytics
3. Data Science Methodology
4. Python Project for Data Science
